# README
 Ruby, Railsで学習した内容のアウトプットのため、HTMLで返す記事投稿アプリを作成しました。
 ※Railsで実装するアプリケーションの全体像を把握するために、JSONで返すアプリ（Qiita風アプリ）の前に作成しました。
 ※自身ではバックエンド部分の実装を行い、フロントエンド部分は用意されていたものを使用してつなぎ合わせました。

# 動作環境
- Ruby: 3.1.3
- Rails: 6.1.7.1
- Node 14.21.3
- DB: PostgreSQL

# 機能一覧
- ユーザー登録、ログイン機能（devise）
- 記事一覧取得機能
- 記事投稿機能
- マイページ機能
- 記事のタグ付け機能
- 検索機能
- ページネーション
